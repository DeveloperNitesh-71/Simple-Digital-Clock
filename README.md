# Digital Clock

A simple digital clock web application that displays the current time in real time.
This project focuses on JavaScript timing functions and dynamic DOM updates with a clean and minimal user interface.

---

## 🚀 Features

* ⏰ Real-time clock display
* 🔄 Automatic time updates every second
* 🎨 Simple and minimal UI
* ⚡ Lightweight and fast
* 📱 Responsive layout

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* JavaScript Date Object

## 🧠 How It Works

* JavaScript retrieves the current time using the `Date` object.
* The time updates every second using `setInterval()`.
* The DOM dynamically refreshes to display the latest time.
